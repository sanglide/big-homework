### 退票策略修改(TicketController)

| 用例             | 新增退票策略              |
| ---------------- | ------------------------- |
| 类型：           | postRequest               |
| url:             | /ticket/refund/add        |
| 前端传给后端的   | TicketRefundVO            |
| 后端返回给前端的 | ResponseVO.buildSuccess() |
| 前端             | 张亦池                    |
| 后端             | 蒋立志                    |

TicketRefundVO属性:

- rate:double //价格折算比率,范围为(0,1),如果可以退票则计算,不能则退票失败
- limitHours:int //距离拍片时间不得退票时间(单位:小时)
